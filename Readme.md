# Macaulay2Web - a Web App for Macaulay2

Macaulay2Web is a web interface for the [Macaulay2](http://www.macaulay2.com).

It is currently running at https://www.unimelb-macaulay2.cloud.edu.au/.

# HTTPS

Warning! This branch is obsolete! It has been superseded by the `main` branch.
Just run
```bash
M2WEB_HTTPS_MODE=greenlock npm start docker 80 443
```
or change the configuration file `src/server/defaultOptions.ts` to make it the default.
